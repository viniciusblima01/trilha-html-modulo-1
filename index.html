História dos computadores

Os primeiros computadores surgiram durante a Segunda Guerra Mundial para fazer cálculos militares. Eram enormes, ocupavam salas inteiras e funcionavam com válvulas, como o famoso ENIAC (Electronic Numerical Integrator And Computer, que em português quer dizer Computador e Integrador Numérico Eletrônico) , criado em 1946.

O ENIAC, primeiro computador eletrônico digital de grande porte, foi apresentado em 14 de fevereiro de 1946 pelos engenheiros John Eckert e John Mauchly, nos Estados Unidos. Criado para o exército durante a Segunda Guerra, ele pesava 30 toneladas, ocupava 180 m², e consumia muita energia. Seu funcionamento dependia de 18 mil válvulas e cartões perfurados operados por mulheres, consideradas as primeiras programadoras.

Com o tempo, as válvulas foram substituídas por transistores, que deixaram os computadores menores, mais rápidos e confiáveis. Depois vieram os circuitos integrados e, mais tarde, os microprocessadores, que permitiram a criação dos computadores pessoais.

Hoje, os computadores estão por toda parte: no celular, no carro, na TV e até em eletrodomésticos, muito mais potentes do que os primeiros gigantes da história.

Apesar de ter desempenho inferior a qualquer calculadora atual, o ENIAC foi revolucionário, realizando mais cálculos em 10 anos do que toda a humanidade até então. Após sua desativação, suas peças foram espalhadas por museus. A computação pessoal só surgiria décadas depois.

O que são Clients e Servers?

O funcionamento da internet é baseado na comunicação entre clientes e servidores. O cliente (client) é quem faz os pedidos, como quando usamos um navegador para acessar um site. Já o servidor (server) é um computador que armazena os dados e responde a esses pedidos, enviando arquivos como textos, imagens e vídeos. Dessa forma, sempre que acessamos uma página, o cliente solicita as informações e o servidor entrega, permitindo que o conteúdo apareça na tela.

Cliente → Pede: "Quero abrir o site X."
Servidor → Responde: "Aqui estão os arquivos do site X."
Cliente → Mostra na tela.

O que é HTML e como é uma estrutura básica de uma página em HTML?

Primeiramente, o que é HTML?


HTML é uma abreviação para HyperText Markup Language, é a linguagem usada para criar e estruturar páginas da internet, mas não é considerada uma linguagem de programação. Ela serve para organizar e estruturar os conteúdos, como textos, imagens, links, listas, tabelas, vídeos, etc.

Mas qual é a diferença?

HTML: É uma linguagem de marcação usada para montar a estrutura de sites.
Linguagens de programação: São linguagens como JavaScript, que permitem adicionar interatividade e tornar as páginas mais dinâmicas.

O HTML (HyperText Markup Language) utiliza tags para organizar o conteúdo da página, como textos, títulos, listas, links e imagens.
Ele funciona como o esqueleto do site, definindo onde cada item vai aparecer. Por outro lado, linguagens de programação como JavaScript permitem criar interações com o usuário, modificar elementos da página e adicionar comportamentos, como botões que mudam de cor, pop-ups, ou animações.
Trabalhando juntos, o HTML (estrutura), o CSS (responsável pela aparência e design) e o JavaScript (funcionalidade e interatividade) formam páginas modernas, dinâmicas e atrativas. Vale destacar que, apesar de o HTML ser essencial para criar páginas básicas, ele não executa cálculos, nem toma decisões, como fazem as linguagens de programação. Sua função é exclusivamente estruturar o conteúdo.

Qual estrutura básica de uma página em html?:

A estrutura básica de um arquivo HTML é como um modelo inicial que todo site precisa ter para funcionar corretamente no navegador. Abaixo podemos ver um exemplo, onde as Tags foram marcadas de azul:

<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Document</title>
  </head>
  <body>
    <h1>Ola Tudo Bem??</h1>
    <p>Aqui fica tudo que sera mostrado na pagina web </p>
  </body>
</html>

O que são tags?

Tags são comandos do HTML que marcam e organizam o conteúdo da página. Elas definem o que é texto, título, imagem, parágrafo, etc. Abixo alguns exemplos de Tags comumente usadas.

Tag <h1> até <h6>

Definem títulos, sendo <h1> o mais importante e <h6> o menos.

Exemplo:

<h2>Título 2</h2>
<h3>Título 3</h3>
<h4>Título 4</h4>
<h5>Título 5</h5>
<h6>Título 6</h6>

Resultado:

Título 1

Título 2

Título 3

Título 4

Título 5
Título 6

Tags <ol>, <ul> e <li>

<ol> cria listas ordenadas (numeradas).
<ul> cria listas não ordenadas (com marcadores).
<li> define cada item da lista.

Exemplo lista ordenada:
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

Resultado:

  1. Item 1
  2. Item 2

Exemplo lista não ordenada:
<ul>
  <li>Item A</li>
  <li>Item B</li>
</ul>

Resultado:

  • Item A
  • Item B
